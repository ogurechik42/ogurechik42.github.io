Помощь

Данный раздел представляет собой описание предназначения веб-приложения "Alex.journal" и содержит подробную информацию для всех вкладок.

Загрузка оценок

В данной вкладке Вы можете загрузить имеющийся журнал с оценками в форматах (.сsv, .txt, .xlsx). Первая строка журнал представляет собой шапку. В ней описаны: ФИО, класс, информатика, физика, математика, литература, музыка. Порядок и соответсвтие важны. Это обеспечивает правильный функционал веб-приложения. Для загрузки журнала выберете файл, нажав на соответствующую кнопку. Название файла появится снизу. Далее нажмите "Загрузить". Ниже Вы увидете загруженные значения в табличном представлении.

Создание и редактирование

В этой вкладке можно редактировать загруженные значения, а также добавлять и удалять строки. Поля "Информатика", "Физика" и т.д. принимают только численные значения от 1 до 5. Чтобы сохранить отредактированные значения в файл, нажмите "Скачать таблицу" ниже. Таблица скачается в формате .csv.

Табличная статистика

Данная вкладка представляет собой статистические данные об обучающихся, а именно: средняя оценка, медиана, количество оценок для каждого предмета. Выбор предмета осуществляется из выпадающего списка. Данные берутся из таблицы раздела "Создание и редактирование". Статистика представлена для каждого класса и среди всех учеников для каждого выбранного Вами предмета.

Графическая статистика

Этот раздел отражает статистические данные из таблиц вкладки "Табличная статистика" для каждого предмета. Выбор предмета осуществляется из выпадающего списка. Данные также берутся из таблицы раздела "Создание и редактирование". Представлены следующие графики: "Средняя оценка по классам", "Медиана по калссам", "Количество оценок", "Распределение оценок".
